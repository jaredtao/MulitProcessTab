## 说明

标题栏和首页，是主进程，每个tab页面，是一个独立的子进程。

子进程窗口通过Windows API  SetParent的方式，附加到主进程。

Tab切换，使用进程间通讯机制，将目标Tab的窗口置顶。

## 进程间通讯机制

使用Qt的LocalSocket功能。

主进程启动之后，创建一个LoaclServer，使用随机uuid作为server名称。

创建Tab时启动子进程，将server名称传递给子进程，子进程启动后，创建LoclaSocket来连接server。
